# OokExtension

This project aims to package up a bunch of language extensions for Visual Studio - 

the idea to allo developers : 

1. To Create a Ook Project.
2. To ALlow Developers to Add Ook files to the project
3. Add Intellisense / Code Completion to Ook files
4. Add Ook custom icon for Solution Explorer
5. Allow Ook files to be interpretted by a class file (allowing unit testing of Ook files etc).
